# Estruturas de Dados em Java — TDA

Este projeto contém a implementação de **estruturas de dados básicas** em Java: **Lista**, **Pilha** e **Fila**, conforme os exercícios propostos pelo **Prof. Odair Gabriel** na disciplina de **TDA - Estruturas de Dados em Java**.

---

## Objetivo

Implementar classes que utilizem vetores de inteiros para simular estruturas de dados clássicas, aplicando conceitos fundamentais de **inserção**, **remoção** e **visualização de dados**.

---

## Exercícios

### 1. Classe Lista
- Usa um vetor de inteiros para armazenar até **5 elementos**.  
- Métodos implementados:
  - `inserir(int valor)` → insere um valor na lista.
  - `mostrar()` → exibe todos os elementos armazenados.

---

### 2. Classe Pilha
- Permite **empilhar** e **desempilhar** valores inteiros.  
- Métodos implementados:
  - `empilhar(int valor)` → adiciona um elemento no topo da pilha.
  - `desempilhar()` → remove o elemento do topo da pilha.
  - `mostrar()` → exibe o conteúdo atual da pilha.

---

### 3. Classe Fila
- Utiliza um vetor de inteiros para simular uma **fila** de até **5 posições**.  
- Métodos implementados:
  - `enfileirar(int valor)` → adiciona um elemento no final da fila.
  - `desenfileirar()` → remove o primeiro elemento da fila.
  - `mostrar()` → mostra o estado atual da fila.

---

### 4. Classe Principal
- Contém um **menu interativo** para testar as classes `Lista`, `Pilha` e `Fila`.  
- Permite ao usuário escolher qual estrutura deseja manipular e qual operação realizar.
